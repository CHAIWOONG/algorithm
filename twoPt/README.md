# 슬라이딩 윈도우(Sliding Window)
 - 슬라이딩 윈도우는 배열을 따라 왼쪽에서 오른쪽으로 고정된 크기의 부분 배열을 이동하는 알고리즘
 - 배열의 크기가 N, 고정된 크기가 K인 경우 시간복잡도는 O(N+K)이다.
